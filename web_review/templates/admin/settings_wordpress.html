📋 애플리케이션 비밀번호 생성 방법

  1. WordPress 관리자 페이지에 로그인
  2. 사용자 > 프로필 이동
  3. "애플리케이션 비밀번호" 섹션에서 새 비밀번호 생성
  4. 생성된 비밀번호를 복사하여 아래에 붙여넣기
WordPress가 설치된 URL ( trailing slash 없이)
WordPress 로그인 사용자명
프로필 > 애플리케이션 비밀번호에서 생성
